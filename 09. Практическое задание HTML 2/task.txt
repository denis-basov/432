1. Создайте форму. 
2. В форме создайте элементы для ввода имени, фамилии, электронного адреса, пароля, а также описание к этим полям.
3. После поля с паролем в форме добавьте элемент выбора питомца используя радио кнопки.
4. В форме добавьте элемент выбора питомца пола используя чекбокс.
5. Далее в форме добавьте выпадающий список, состоящий из произвольных элементов.
6. Добавьте ранее созданному списку возможность множественного выбора значений.
7. Ниже в форме добавьте текстовое поле размером 5 строк и 40 колонок.
8. После текстового поля добавьте элемент для прикрепления файла.
9. Последним элементом формы добавьте кнопку ее отправки.
10. Создайте ненумерованый список с произвольными элементами.
11. Создайте нумерованый список с римскими цифрами в нумерации.
12. Создайте блок (DIV) с работниками (животными, авто, ...) внутри каждого блока должна быть картинка, заголовок, параграф, список.
