1. INT - целые числа (1, 5, 8, 2, 45, 78)
2. TEXT - текстовый тип данных, произвольный текст большого размера.
3. VARCHAR(кол-во) - строка переменной длины до 255 символов.
4. FLOAT - число с плавающей точкой ( 34.99, 99.99 )


Атрибуты столбцов
NOT NULL - значение в столбце не может отсутствовать
PRIMARY KEY - задаем первичный ключ
AUTO_INCREMENT - СУБД увеличивает значение в столбце самостоятельно
